# Project: Tic-Tac-Toe Game (ttt)

This project is a web-based Tic-Tac-Toe game.

## Tech Stack
- **Framework:** React
- **Language:** TypeScript
- **Styling:** Vanilla CSS (preferred for flexibility)
- **Testing:** Vitest

## Coding Standards
- Use functional components with hooks.
- Prefer explicit type definitions over `any`.
- Keep components small and focused.
- Write unit tests for game logic (e.g., win conditions) using Vitest.
- Follow a clean, modern UI design.

## Project Structure
- `/src/components`: UI components
- `/src/logic`: Game engine and logic
- `/src/__tests__`: Test files
